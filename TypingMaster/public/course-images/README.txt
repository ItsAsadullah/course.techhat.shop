# Course Images

Place course cover images in this folder.

Expected file names (set in `src/context/CourseContext.tsx`):
- english.jpg  → English Typing card
- avro.jpg     → Bangla Avro card
- bijoy.jpg    → Bangla Bijoy card

Then update the `coverImage` field in the COURSES array:

  coverImage: "/course-images/english.jpg",

Supported formats: JPG, PNG, WebP
Recommended size: 600×280px (2:1 ratio)
